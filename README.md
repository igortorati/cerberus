# Welcome to Cerberus!
This is a Discord Bot created to control players entering and leaving TTRPG games on Covil do Lich.
This bot is configured to run on Cloudflare Workers and is configured to automatically be rebuild when the master branch is updated.
To execute this code locally you can execute the following commands:
```
npm i
npm run start
npm run ngrok
```
You will need to have Ngrok and the Cloudflare Wrangler in your machine to execute locally.

# About Covil do Lich
Covil do Lich is a brazilian Discord server for finding comissioned TTRPG games.
Check out our server [here](https://discord.com/invite/covildolich).
